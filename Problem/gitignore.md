### .gitignore이란??

- .gitignore파일이란 Git 버전 관리에서 제외할 파일 목록을 지정하는 파일이다.
- 파일 혹은 폴더 목록을 지정하여 작성하면, 특정 목록들은 git의 관리대상에서 제외된다.
